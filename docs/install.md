# Install

## Clone it from github

```bash
git clone https://github.com/swwind/roigu.git
```

## Set cnofigs

1. Copy `_config.template.yml` to `_config.yml`.

2. Modify your `_config.yml`.

3. publish

   ```bash
   hexo clean && hexo d -g
   ```

Perfect! Now you have completed the installation!
